# Weather-logs-SQLite
Scraping weather data for Cairo and save it in SQLite database

#Database Design

- Temp: int
- wind: text
- humidity: int
- visibility: int
- pressure: int
- time_now: datetime

# Data Source:
https://eg.freemeteo.com/weather/cairo/current-weather/location/?gid=360630&language=english&country=egypt
